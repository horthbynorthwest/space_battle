A simple locally run, 2 player pygame application.

## Quick start!
* clone the code
* ``cd space battle`` and run ``pip3 install -r requirements.txt``
* ``python3 main.py``
* Play the game!

## Controls

### Yellow Space Ship:
* movement = 'w', 'a, 's', 'd' keys
* shooting = left shift key

### Red Space Ship:
* movement = arrow keys
* shooting = right shift key
